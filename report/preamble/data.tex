%% Abstract
\newcommand*{\Abstract}{%
    This report studies the interaction between randomized sketching and mixed-precision arithmetic in scaled dot-product attention. The central approximation replaces the exact logit matrix $QK^T$ by a sketched surrogate of the form $QSS^TK^T$, where $S$ is a random projection. The goal is to compare sketching error and finite-precision error, and to determine when mixed precision remains reliable once randomized approximation is already present. The accompanying Python experiments measure logit error, attention-weight error, output error, and runtime across several sketch sizes and precision modes.
}
